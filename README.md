# CineIndex

## Objetivo

CineIndex é uma aplicação web que permite aos usuários organizar e gerenciar um catálogo pessoal de filmes. Com ela, você pode manter um registro de filmes que deseja assistir, que está assistindo, ou que já assistiu, além de marcar seus filmes favoritos de forma prática e intuitiva.

A plataforma oferece uma forma simples e eficaz de acompanhar sua jornada cinematográfica, mantendo informações sobre os filmes em um único lugar.

## Tecnologias Utilizadas

- **React** - Biblioteca para construção da interface
- **TypeScript** - Linguagem para melhor tipagem e segurança
- **Vite** - Ferramenta de build e desenvolvimento
